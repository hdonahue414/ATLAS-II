# ATLAS v20 Clean Architecture

ATLAS has been normalized into six decision-useful sections:

1. Core
2. Evidence
3. Fit
4. Decision
5. Visuals
6. Development

Removed from visible navigation:
- All systems scaffolding
- Mega indices as a top-level gimmick
- Redundant cards/radar/topology/timeline pages
- Overly speculative tooling as separate pages

Kept but consolidated:
- Relationship tracker
- Public testimony
- Alumni/current outcomes
- Evidence hierarchy
- Documentary philosophy
- Daily-life projection
- Sustainability stress tests
- Identity preservation
- Forecasts
- Bias controls
- Attraction/regret
- Heatmap/confidence/location/portfolio
- Application/portfolio/thesis tools
