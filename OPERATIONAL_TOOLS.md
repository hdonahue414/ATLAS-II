# Operational Tools v12

This version makes the final layer work immediately in-browser.

Working modules:
- Decision pressure detector architecture
- Application alignment by school
- Portfolio gap ranking
- Thesis feasibility simulator
- Collaborator compatibility map
- Rejection recovery model
- Career economics pressure calculator
- Relationship/life integration calculator
- Attraction decomposition by school
- Regret-risk map by school
- Stop-researching detector
- Data editor/export

Limitations:
- Static dashboard; no backend.
- Calculators are decision-support heuristics, not validated predictive models.
- Values can be edited directly in data.json.
